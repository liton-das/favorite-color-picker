# 🎨 Favorite Color Picker

A simple and interactive web-based color picker that allows users to select and preview colors using RGB sliders and HEX input. Built with HTML, CSS, and JavaScript, this tool provides real-time feedback and a smooth user experience with additional sound effects.

---

## 🚀 Features

- 🔧 RGB Sliders (Red, Green, Blue)
- 🎯 Real-time Color Preview Box
- 🟣 HEX ↔ RGB Conversion
- 🎵 Sound effects on user actions
- 🎉 Win conditions and validation logic
- 🔁 Reset / Replay functionality
- 📱 Responsive UI design

---
## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## ⚙️ How to Use
1. Adjust the RGB sliders or input a HEX code.

2. Watch the color preview box update instantly.

---

## 🌈 Live Features

* Real-time display of the current color
* HEX and RGB values update as you interact
* Win condition with sounds if matched
